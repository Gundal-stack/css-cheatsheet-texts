CSS Cheatsheet - Text transformation

Property Name Example Syntax Description
color color: #FF0000; Sets the color of the text.
font-size font-size: 16px; Sets the size of the text.
font-family font-family: Arial, sans-serif; Sets the font family of the text.
font-weight font-weight: bold; Sets the weight of the text.
text-align text-align: center; Sets the alignment of the text.
text-decoration text-decoration: underline; Adds decorations to the text.
text-transform text-transform: uppercase; Transforms the case of the text.
line-height line-height: 1.5; Sets the height of each line of text.
letter-spacing letter-spacing: 1px; Sets the spacing between letters in the text.
word-spacing word-spacing: 2px; Sets the spacing between words in the text.
